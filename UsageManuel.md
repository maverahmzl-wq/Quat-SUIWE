# Usage Manual for SUIWE

1. Unzip the archive and explore the `widgets` folder.
2. Include the headers in your C/C++ project.
3. Use the provided `draw()` methods as placeholders to render your UI elements.
4. Add your own scripts and functionalities as needed.
5. Compile the project with GCC/MSYS or your preferred C/C++ compiler.
6. Bin folder contains example DLL/EXE placeholders.
