# SUIWE - Version 1.0.0

SUIWE (Shortcut UI Widget Extensions) is a lightweight framework to help developers integrate UI elements into their C/C++ projects.

Features:
- Predefined widgets like button, text, input, checkbox, toggle switch, frame, scrolling frame, color palette, dropdown, image.
- Easy integration with existing projects.
- Placeholder source files for fast prototyping.
